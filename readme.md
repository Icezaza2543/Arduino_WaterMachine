# ตู้กดน้ำดื่มอัตโนมัติด้วย Arduino (Arduino Water Vending Machine)

## 📌 ภาพรวมโปรเจคต์ (Project Overview)
โปรเจคต์นี้คือระบบควบคุมตู้กดน้ำดื่มหยอดเหรียญอัตโนมัติที่พัฒนาด้วยบอร์ดไมโครคอนโทรลเลอร์ Arduino โดยมีจุดเด่นในเรื่องของการควบคุมการจ่ายน้ำตามจำนวนเงินที่หยอด ควบคุมระบบเปิด-ปิดไฟส่องสว่างอัตโนมัติตามเวลาที่กำหนด และมีหน้าจอแสดงผลสถานะการทำงานต่างๆ ให้ผู้ใช้งานทราบ

### ฟีเจอร์หลักในระบบ
* **Coin Acceptor Integration**: ทำงานผ่านระบบ Hardware Interrupt รองรับการรับเหรียญและการนับยอดเงินแบบเรียลไทม์
* **RTC Time Control**: โมดูลรับเวลาแบบ Real Time Clock (RTC) ใช้สำหรับควบคุมระบบไฟส่องสว่างหน้าตู้ (เปิดตอน 18:00 น. และปิดตอน 06:00 น.)
* **LCD Display**: ใช้จอ LCD แบบ I2C แบล็คไลท์สำหรับแสดงผล เวลา, จำนวนเงิน (Baht), และปริมาณน้ำที่รองรับ 
* **Relay Management**: ระบบรีเลย์ 4 ช่อง สำหรับควบคุมการทำงานของปั๊มน้ำ (Water Pump) และชุดไฟสถานะ
* **Water Tank Tracking**: ฟังก์ชันในการคำนวณและติดตามปริมาณน้ำที่เหลืออยู่ในถังพัก (อ้างอิงจาก `CONF.h` และการสื่อสารผ่าน Serial ข้อมูลส่งให้ ESP-01)
* **EEPROM Storage**: อ่านและบันทึกประวัติ หรือตั้งค่าระบบลงในหน่วยความจำถาวร

## ⚙️ Workflow การทำงาน (Workflow)

1. **โหมดสแตนด์บาย (Standby Mode)**
   * หน้าจอ LCD จะทำหน้าที่แสดงเวลา ณ ปัจจุบัน 
   * โค้ดจะคอยตรวจสอบเวลาผ่านเซ็นเซอร์ RTC หากอยู่ในช่วงเวลา 18:00 ถึง 06:00 น. ระบบจะสั่งเปิดรีเลย์ดวงไฟหน้าตู้แบบอัตโนมัติ

2. **โหมดรับเหรียญ (Coin Insertion Mode)**
   * เมื่อผู้ใช้ทำการหยอดเหรียญ ระบบช่องรับเหรียญจะส่งสัญญาณพัลส์เข้าขาขัดจังหวะ (Interrupt) ของ Arduino 
   * นำสัญญาณจำนวนพัลส์มาคำนวณและสะสมเป็นยอดเงิน 
   * หน้าจอจะแสดงอัปเดตยอดเงินที่หยอด (`Baths`) พร้อมแสดงปริมาณน้ำฝั่งของเหลว (`liquid`) ตามสัดส่วนของเงิน

3. **โหมดจ่ายน้ำ (Dispensing Mode)**
   * เมื่อผู้ใช้กดปุ่ม / สวิตช์ (`sw`) ระบบจะคำนวณจำนวนเงินที่หยอดมาให้เป็นระยะเวลาการทำงานของปั๊มน้ำ (มิลลิวินาที)
   * ระบบจะสั่งปิดและเปิดรีเลย์ปั๊มน้ำ (`R[3]`) ตามจำนวนวินาทีที่ได้คำนวณไว้ โดยให้ไฟสถานะดวงอื่นๆ สว่างขึ้นเพื่อแจ้งเตือนว่ากำลังดำเนินการจ่ายน้ำ

4. **โหมดเช็คสถานะและรีเซ็ต (Status Monitoring Mode)**
   * ระบบจะบันทึกปริมาณน้ำที่ค่อย ๆ ถูกจ่ายออกไปเทียบกับความจุถัง (ค่า `tankvalue` = 5000) 
   * ทำการอัปเดตและคำนวณ `leftvalue` ของน้ำที่เหลือ เพื่ออาจส่งค่านี้แจ้งเตือนระดับน้ำใกล้หมด และรีเซ็ตค่าสถานะหน้าจอกลับเข้าสู่โหมดสแตนด์บาย

## 💰 การคิดราคา (Pricing)

ระบบการคิดราคาใช้วิธีแปลง "ยอดเงินบาท (Baht)" เปลี่ยนเป็น "ปริมาณของเหลว (Liter)" และแปลงเป็น "หน่วยเวลาของปั๊ม (Milliseconds)"

* **ตรรกะเบื้องต้น**: ยอดเงิน `X` บาท = ปริมาตรกี่ลิตร -> สั่งให้ปั๊มทำงานเป็นระยะเวลากี่มิลลิวินาที (Time-based Pump Control)
* ตัวบ่งชี้จากโค้ดได้กำหนดไว้คร่าวๆ ว่า `1 liter = 5000 millis()` 
* ปริมาณน้ำและเวลาการสูบจะถูกคำนวณตามสัดส่วนคณิตศาสตร์ เช่น `liters = Baths * Factor` และนำไปหน่วงเวลาผ่านการอ่านค่าของ `millis()` ทำให้ระบบจ่ายน้ำมีความแม่นยำสูง

## 📚 แหล่งอ้างอิง (References)

ซอฟต์แวร์และไลบรารีที่จำเป็นในการคอมไพล์สำหรับโปรเจคต์ Arduino ตัวนี้:
* **Wire Library:** สำหรับเชื่อมต่อการสื่อสาร I2C
* **EEPROM Library:** สำหรับบันทึกสถานะการตั้งค่าจำนวนตัวแปรลง ROM 
* **RTClib:** โดย [Adafruit] ใช้สำหรับอ่านและและตั้งค่าโมดูลเวลา Real-Time Clock
* **LiquidCrystal_I2C:** ใช้สำหรับควบคุมลอจิกการแสดงผลบนหน้าจอ LCD
* Document การรับพัลส์จากกระปุกหยอดเหรียญ (Coin Acceptor Documentation & Interrupt Pins mapping)
